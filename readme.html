Strongman Counter — Carnival Tower

Overlay + remote controller for OBS (Browser Source). Premium Carnival-Strongman design with buzzer animation at goal.

  1. Upload this repo to GitHub and enable Pages (main branch, root).
  2. In OBS, add a Browser Source with URL to index.html?room=NAME. Multiple rooms are isolated.
  3. Open controller.html. Enter the same room and press Connect.

Features

  • Custom goal, milestones, unit label
  • Quick-add buttons (+10 → +200), custom input, undo/reset
  • Buzzer light animation at goal
  • Local persistence per room; keyboard shortcuts

URL Params (Overlay)

index.html?room=legday

Keyboard Shortcuts (Controller)

1..0=+10..+100, Shift adds +110..+200, Z undo, R reset.
